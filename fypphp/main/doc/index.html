Installation

After you have finished creating your site with our theme, all you need to do is upload the contents of your project onto your hosting server via FTP.

You can learn more about this here.

Background

  1. Find div with "middle" id
  2. Add bottom bold html above div

Youtube background

  1. Add "youtube-background" class to body tag
  2. Open the file "*.html"

                    
                  <div id="middle">
                    <div>
                      <div class="bg">
                        <a class="player" data-property="{videoURL:'SHwTTW_6_eU', containment:'.bg', autoPlay:true, mute:true, quality: 'small', startAt:0, showControls:false, opacity:1}"></a>
                      </div>
                      ...
                    </div>
                  </div>
  3. What you can pass to the “videoURL” paramenter:

    To retrive the video from YouTube you can use any of the following string:

    • The URL of the page containing the video
      videoURL: ‘https://www.youtube.com/watch?v=V2rifmjZuKQ’.
    • The short URL available fron the YouTube share panel
      videoURL: ‘http://youtu.be/V2rifmjZuKQ’.
    • The video ID
      videoURL: ‘V2rifmjZuKQ’.
  4. Add "jquery.mb.YTPlayer.min.js" to bottom of body

Image slideshow background

  1. Open the file "*.html"

                    
                  <div id="middle">
                    <div>
                      <div id="slider" class="bg"></div>
                      ...
                  </div>
  2. Open the file "js/main.js"

                    
                  $('#slider').vegas({
                    slides: [{
                      src: '../images/1920x1280.png'
                    }, {
                      src: '../images/1920x1280.png'
                    }, {
                      src: '../images/1920x1280.png'
                    }, {
                      src: '../images/1920x1280.png'
                    }]
                  });
  3. Add "vegas.css" to head tag
  4. Add "vegas.js" to bottom of body

Single image background

  1. Open the file "*.html"

                    
                  <div id="middle">
                    <div>
                      <div id="single" class="bg"></div>
                    ...
                  </div>
  2. Open the file "css/index.css"

                    
                  .bg#single {
                    background: url("../images/1920x1280.png") no-repeat center center fixed;

Animation

  1. Find div with "middle" id
  2. Add bottom bold html below div
  3. Add scripts to bottom of html

Constellation animation

            
                <div id="middle">
                  <div>
                    ...
                    <canvas id="constellation"></canvas>
                  </div>
                </div>
                ...
                <script src="../js/animation/TweenLite.min.js"></script>
                <script src="../js/animation/EasePack.min.js"></script>
                <script src="../js/animation/rAF.js"></script>
                <script src="../js/animation/constellation.js"></script>
              </body>

Low-poly animation

            
                <div id="middle">
                  <div>
                    ...
                    <canvas id="lowpoly"></canvas>
                  </div>
                </div>
                ...
                <script src="../js/animation/rAF.js"></script>
                <script src="../js/animation/lowpoly.js"></script>
              </body>

Starfield animation

Add script to head 

            
                <script src="../js/animation/starfield.js"></script>
              </head>
              <body>
                ...
                <div id="middle">
                  <div>
                    ...
                    <canvas id="starfield"></canvas>
                  <div>
                </div>

Color

There are 4 color options (blue, green, grey, orange). You can change it.

  1. Open the html file
  2. You will find in head:

                    
                  <link rel="stylesheet" type="text/css" href="../css/colors/green.css" />

Contact form

  1. Open php/contact.php

    $to = 'myemail@mail.com';
